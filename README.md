У межах даної роботи було реалізовано клас Ellipse, який моделює геометричну фігуру — еліпс.

Клас містить конструктор з параметрами semiMajorAxis та semiMinorAxis, гетери і сетери, а також метод toString().

Реалізовано методи для обчислення основних характеристик еліпса:
area() — площа еліпса,
majorAxisLength() — довжина великої осі,
minorAxisLength() — довжина малої осі,
circumferenceApprox() — приблизна довжина еліпса,
eccentricity() — ексцентриситет еліпса.

Для математичних обчислень використано константу Math.PI.

Для перевірки коректності роботи створено тестовий клас EllipseTest з використанням JUnit 4, який містить юніт-тести для перевірки роботи реалізованих методів. У тестах використано анотацію @Test та метод assertEquals() з допустимою похибкою 0.01 для обчислень типу double.
